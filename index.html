Description:

This console-based Student Attendance System is designed to manage attendance records for multiple classes effectively. It allows the user to:

1. Add Attendance Records: Start a new attendance record for a class, including the class name, date, and student names.

2. View Attendance: Display attendance details for all or specific classes, including student lists.

3. Update Attendance: Modify the attendance records for an existing class, replacing the old list with a new one.

4. Delete Attendance: Clear all attendance records in one action.

Specialties:

  • Ease of Use: A simple and intuitive menu-driven interface with numbered options.
  • Scalability: Handles up to 10 classes, with each class capable of having up to 40 students.
  • Error Handling: Includes basic safeguards, such as checking for invalid inputs or ensuring class capacity is not exceeded.
  • Flexibility: Allows dynamic updates to attendance records without restarting the program.
